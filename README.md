StreamProbe is a frontend-only, Manifest V3 Chrome Extension designed for OTT engineers, streaming developers, and video infrastructure teams. It silently hooks into browser network traffic to intercept, parse, and analyze streaming manifests (HLS/DASH) and DRM license requests in real-time.

---

## 🚀 Features

- **📡 Live Stream Detection:** Automatically intercepts `.m3u8`, `.mpd`, and media segments (`.ts`, `.m4s`, `.mp4`).
- **🔐 DRM Analysis & PSSH Extraction:** Intelligently detects Widevine, PlayReady, and FairPlay requests. Deeply parses DASH (`<cenc:pssh>`) and HLS (`#EXT-X-SESSION-KEY`) manifests to automatically extract and list base64 PSSH boxes!
- **📜 Beautiful Manifest Explorer:** Parses raw XML/M3U8 playlists into a stunning, human-readable hierarchy of Periods, Video/Audio Tracks, Bitrates, Codecs, and clearly badges tracks protected by DRM.
- **🛡️ 403 CDN Bypass Architecture:** Uses Chrome's native DevTools `onRequestFinished` stack to pull manifest data natively, completely bypassing Cloudflare bots, CORS blocks, and 403 Forbidden errors! 
- **🔄 Hot-Reloading & Auto-Clear:** Hooks into page navigation. When you refresh a video page, the dashboard automatically wipes old history and starts fresh like a true dev tool.
- **🔍 Deep Network Inspection:** View full request/response headers and POST payloads (Base64). Mark any missed POST request manually as a DRM License!
- **⏯️ On-Demand Replay:** Instantly replay any intercepted DRM/network request with a single click and view the raw response payload in the UI on-demand to preserve extension performance.
- **💻 Quick-Glance Popup UI:** Click the extension icon to instantly see active playback info, DRM types (Widevine/PlayReady), and extracted Video/Audio/PSSH stats without even opening DevTools!
- **💾 Data Export:** Export the entire session's network telemetry to JSON for offline debugging and bug reporting.
- **💅 Futuristic UI:** Built with React, Tailwind CSS (v4), and Framer Motion for a buttery-smooth, dark-mode engineering aesthetic.

### Option 2: Developer Mode (Local Installation)

1. **Clone the repository:**

   ```bash
   git clone https://github.com/SnehKr/streamprobe-extension.git
   cd streamprobe-extension
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Build the extension:**

   ```bash
   npm run build
   ```

4. **Load into Chrome:**
   - Open Google Chrome and navigate to `chrome://extensions/`.
   - Enable **Developer mode** in the top right corner.
   - Click **Load unpacked**.
   - Select the `dist/` directory generated by the build process.

## 💻 Usage Guide

1. Open Chrome Developer Tools (`F12` or `Ctrl+Shift+I`).
2. Navigate to the new **StreamProbe** tab.
3. Open a streaming service or a demo player (e.g., [Bitmovin DRM Demo](https://bitmovin.com/demos/drm) or [Shaka Player](https://shaka-player-demo.appspot.com/)).
4. As the stream loads, StreamProbe will automatically populate the dashboard.
5. **Explore:**
   - **Dashboard:** View overall stream health, active streams, and DRM counts.
   - **Network Tab:** Click on any request to expand its deep analysis view (Headers + Payload). Click **Replay** to refetch the request and analyze the response!
